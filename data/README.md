# Data Directory

## Files in This Repository

### `gss24_attitudes_opinions.csv`
Authoritative registry with inclusion decisions, coalesce logic, and binarization rules for the 139 outcome variables analyzed in the paper. Columns include:

- `var` — GSS variable name (uppercase)
- `category` — Fine-grained topic label
- `var_variant` — Comma-separated names of split-ballot or wording experiment variants
- `variant_notes` — Description of the variant type and expected effect size
- `analytic_core` — Binary flag (1 = included in analysis, 0 = excluded)
- `domain` — One of 8 thematic domains (e.g., "Religion", "Race & Inequality"); blank for excluded vars
- `coalesced` — 1 = variants coalesced into main variable before binarization; 0 = not coalesced
- `coalesce_notes` — R expression used to coalesce variants
- `binarized_logic` — Binarization method: `binary`, `3-cat`, `4-cat`, `5-cat`, or `7-cat`
- `binarize_notes` — `to0`/`to1` values for binary recodes; cautions for special cases

### `gss20222024_modeweights.dta`
Generated by `02_weights.qmd`. Mode-specific post-stratification weights for 2022 and 2024 GSS respondents. Contains:

- `wtssps_f2fp` — Weight for FTF/Phone respondents (raked to full-sample margins)
- `wtssps_web` — Weight for Web respondents (raked to full-sample margins)

Use these weights when analyzing 2022 or 2024 GSS responses separately by mode. Merged onto the analysis dataset in `04_mode_gaps.qmd` and `05_persistence.qmd`.

## Files You Must Provide

### `gss7224_r3.dta`
**The raw GSS Cumulative Data File (1972–2024), Release 3 — you must download this yourself.**

GSS data cannot be redistributed. To replicate the analysis:

1. Go to [https://gss.norc.org/get-the-data/stata](https://gss.norc.org/get-the-data/stata)
2. Download the **GSS Cumulative Data File (1972–2024), Release 3** (Stata format)
3. Place the `.dta` file in this directory and rename it to `gss7224_r3.dta`

The path is defined in `00_setup.R`:

```r
GSS_DTA_PATH <- "data/gss7224_r3.dta"
```
